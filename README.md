# LogShield

Privacy-first log sanitizer.
Deterministic. Auditable. Runs locally.

## What this repo contains
- Rust core engine (production-ready)
- Local-only web demo (static, privacy-first)

## Principles
- No logs leave the user's machine
- Same engine everywhere (Web, CLI, CI)
- Deterministic output
